Image URL tester 🚀

Paste an image URL (backend /uploads or any image) and click Preview. The tool will try to fetch the file first to surface HTTP errors.

Examples you can try:
  • http://localhost:8080/uploads/<filename> — URL trả về từ upload API
  • /uploads/<filename> — relative path (may need host)
  • https://via.placeholder.com/600x300 — public test image
